# Learner-Activity

- Complete the application taught in this module with the implementation of shared preferences, along with a button to log out the application at the bottom of the Avengers page. 
- You need to take care that you clear the preferences when you log out of the application. In order to clear the preferences, just use the code given below:
   - sharedPreferences.edit().clear().apply()
- Also, think about the fields you can add in the forgot password and registration screens and add clicks to them.
